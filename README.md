# Aircraft Wing Surface Defect Detection using CNN

**Team Number:** 3  
**Team Members:** Erika, Nikita, Danik  
**Date:** 14 April 2026  

## Project Description
A Convolutional Neural Network (CNN) for automatic detection of surface defects (cracks, dents, scratches) on aircraft wings.

## Why CNN?
- CNNs are designed for image classification
- They detect spatial patterns (edges, textures, defects)
- State-of-the-art for visual inspection tasks
